# Domain Authority

```text
Repair Intake
→ Repair Job
→ Diagnosis
→ Internal Repair / Warranty Claim
→ Receive Back
→ Quality Check
→ Deliver to Customer
→ Close Repair Job
```

เมื่อผลจบงานต้องคืนเงินหรือเปลี่ยนสินค้าตามนโยบายการขาย:

```text
Closed Repair Job
→ Start New Sale Return
→ Eligibility / Approval
→ Refund or Exchange
→ Complete Sale Return
```

กฎ: Workflow หนึ่งห้ามเปลี่ยนสถานะของอีก Workflow กลางทาง
